**Многопользовательский Telegram-бот для комплексной автоматизации откликов на hh.ru.**  
От сбора вакансий до генерации персонализированных AI-писем — всё в одном продукте.

> *«Код — лишь средство. Продукт — это ценность, которую получает пользователь.»*

---

## 📊 Подтверждённые метрики

| Метрика | До | После |
|---|---|---|
| Откликов в день | 10 | **1000+** |
| Время на один отклик | 5–10 мин | **~30 сек** |
| Конверсия (шаблонное письмо) | 1% | — |
| Конверсия (AI-письмо) | — | **10%** |
| Конверсия при матче >60% | — | **30%** |

> Данные получены на реальном тестовом пользователе. A/B тест: 100 откликов с шаблонным письмом vs 50 откликов с AI-письмом.

## ✨ Возможности

### 🤖 Автоматический режим
- Бот сам собирает вакансии, генерирует письма и отправляет отклики
- После каждого отклика присылает карточку с названием, компанией, городом и **вероятностью матча**
- Работает без участия пользователя

### 🎯 Ручной режим
- Каждая вакансия приходит как карточка в Telegram с кнопками:
  - ✅ **Откликнуться** — отправить письмо
  - ⏭️ **Пропустить** — пропустить вакансию
  - ✏️ **Редактировать** — изменить письмо свободным текстом
  - 📄 **Полностью** — посмотреть полный текст письма
- Редактирование письма в свободном формате, например:
  - *«Убери упоминание ретейла, оставь только IT-опыт»*
  - *«Добавь что готов к командировкам»*
  - *«Сделай акцент на управлении проектами»*

### 📊 AI-агент Реваз (оценка матча)
- Анализирует каждую вакансию и сравнивает с резюме пользователя
- Выдаёт **числовую оценку совместимости** (вероятность матча в %)
- Цветовые индикаторы: 🟢 >70% / 🟡 40–70% / 🔴 <40%
- Краткий комментарий почему именно такая оценка
- Отклики с матчем >60% дают конверсию **30% в приглашение на собеседование**

### 📄 Алина — анализ и улучшение резюме
- Глубокий AI-анализ резюме: оценка /10, сильные стороны, слабые стороны
- Ключевые слова для прохождения ATS-фильтров
- Генерация **улучшенной версии резюме** с учётом замечаний
- Три варианта действий: заменить текущее / добавить как новое / оставить текущее
- Хранение нескольких резюме — можно выбрать какое использовать перед каждым запуском

### 🔐 Безопасный многопользовательский онбординг
- Авторизация через номер телефона → SMS-код
- Каждый пользователь получает изолированный Chrome-профиль
- Повторная авторизация не нужна — сессия восстанавливается автоматически
- Браузер запускается на сервере, пользователь только вводит данные в Telegram

### 🔗 Гибкость поиска
- По должности (AI подбирает варианты по резюме или вводишь вручную)
- По готовой ссылке hh.ru с любыми фильтрами (зарплата, график, опыт, город)
- Настраиваемый лимит откликов (1–200)

## ⚡ Быстрый старт

### 1. Клонировать репозиторий

```bash
git clone https://github.com/Ryuvair/hh-bot-pro.git
cd hh-bot-pro
```

### 2. Создать виртуальное окружение

```bash
python -m venv .venv

# Windows
.venv\Scripts\activate

# macOS / Linux
source .venv/bin/activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
playwright install chromium
```

### 4. Настроить переменные окружения

```bash
cp .env.example .env
```

Заполни `.env`:

```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
DEEPSEEK_API_KEY=your_deepseek_api_key
```

### 5. Запустить бота

```bash
python main.py
```

## 📱 Как пользоваться

### Первый запуск

1. Напиши боту `/start`
2. Нажми **🔐 Авторизоваться в hh.ru** — откроется браузер на компьютере
3. Введи номер телефона (10 цифр без +7 и 8, например `9060295956`)
4. Введи SMS-код который придёт на телефон
5. Загрузи резюме — отправь текст резюме в чат (минимум 500 символов)
6. Алина проанализирует резюме и предложит улучшения
7. Выбери или введи должность
8. Укажи лимит откликов
9. Нажми **▶️ Запустить отклики**

### Нижнее меню (всегда доступно)

| Кнопка | Действие |
|---|---|
| ▶️ Запустить отклики | Запустить / остановить сессию |
| 🔄 Режим | Переключить авто / ручной режим |
| 📊 Статистика | Посмотреть статистику откликов |
| 📄 Резюме | Управление резюме |
| 📌 Должности | Управление должностями и ссылками |
| 🔗 По ссылке | Запустить отклики по готовой ссылке hh.ru |
| 🔐 Авторизация hh.ru | Авторизоваться или переавторизоваться |

## 🧠 AI-агенты

### Реваз — оценка матча
Анализирует каждую вакансию перед откликом:
- Извлекает ключевые требования из описания вакансии
- Сравнивает с опытом и навыками из резюме
- Возвращает процент совместимости и краткое объяснение
- В авторежиме пропускает вакансии с низким матчем

### Алина — анализ резюме
Двухэтапный процесс:
1. **Анализ** — оценка /10, сильные/слабые стороны, ATS-ключевые слова
2. **Улучшение** — генерирует переработанную версию резюме (только если оценка < 9)

### Генератор писем
- Персонализированное письмо под каждую вакансию
- 3–4 предложения, ≤400 символов
- Без обращений, подписей, контактов
- Пост-обработка: очистка запрещённых фраз, определение пола соискателя

## 📈 Статистика и история

Бот сохраняет историю каждого отклика:

| Статус | Описание |
|---|---|
| `sent` | Отклик успешно отправлен |
| `skipped` | Пропущен пользователем (ручной режим) |
| `revaz_skip` | Пропущен Ревазом (низкий матч) |
| `error` | Ошибка при отправке |

**📊 Статистика** показывает: всего / отправлено / пропущено / ошибок / сегодня + последние 10 успешных откликов со ссылками.

## 🛡️ Безопасность и антидетект

- Рандомизированные паузы между действиями (5–10 сек)
- Имитация поведения реального пользователя через Playwright
- Поддержка SOCKS5 прокси
- Изолированные Chrome-профили — каждый пользователь авторизован отдельно
- Сессии восстанавливаются без повторного ввода SMS

---

## 🔄 Многопользовательская работа

- Каждый пользователь имеет независимую сессию
- Изолированные Chrome-профили в `chrome_profiles/{telegram_id}/`
- Параллельные сессии не влияют друг на друга
- Раздельная история откликов и статистика
